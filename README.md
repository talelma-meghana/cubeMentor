# CubeMentor

Interactive 3D Rubik's Cube Simulator built using React and Three.js.

## Features

- All 12 Rubik's Cube Moves
- Scramble Generator
- Move History
- Undo / Redo
- Orbit Camera Controls
- 3D Rendering with React Three Fiber

## Tech Stack

- React
- Vite
- Three.js
- React Three Fiber

## Installation

```bash
npm install
npm run dev
```
